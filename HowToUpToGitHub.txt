-------------Lần đầu lấy code từ Github-------------
git clone https://github.com/Dchong1211/TechShop.git

-------------Muốn update code lên Github-------------
git add .
git commit -m "Initial commit"
git push

Hoặc chạy file auto_push.bat để update cho lẹ

-------------Muốn lấy code thằng khác update về Github-------------
git pull origin main --rebase

-------------Sửa lỗi khi code bị kẹt khi up lên GitHub-------------
1. Gõ lệnh sau trong terminal: 
        git rebase --continue
1.1. Nếu Git báo không có thay đổi, gõ thêm: 
        git rebase --skip
1.2. Lặp lại --continue hoặc --skip cho đến khi Git báo: 
        Successfully rebased and updated refs/heads/main.
2. Khi xong, kiểm tra lại: 
        git status
2.2. Phải thấy:
        On branch main
        nothing to commit, working tree clean
3. Rồi push lại:
        git push origin main
